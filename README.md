# Portfolio Website
A modern, interactive developer portfolio showcasing full-stack projects and technical expertise.

## Live Demo
 https://portfolio-website-fhxnhp7ve-anugrah-ks-projects-cc3fdb90.vercel.app

---

## Description
A responsive personal portfolio designed to highlight projects, technical skills, and professional background.  
Built with smooth animations and a clean UI to help recruiters and collaborators quickly explore the author’s work, download the resume, and get in touch via a functional contact form.

---

## Features
- Dynamic project showcase rendered from a centralized data file
- Smooth UI animations using Framer Motion
- Functional contact form powered by Web3Forms
- Fully responsive, mobile-first design with Tailwind CSS
- Animated hero section with typewriter effects
- Direct resume download link

---

## Tech Stack
- **Frontend:** React 19, Vite, Tailwind CSS 4
- **Animations:** Framer Motion
- **Routing:** React Router DOM 7
- **Libraries:** Lucide React, Swiper, Typewriter-effect
- **Other:** Web3Forms (Email API)
