Project Overview

A full-featured e-commerce platform where users can browse products, manage shopping carts, and complete secure purchases. Built from scratch using the MERN stack (MongoDB, Express.js, React, Node.js), this project demonstrates end-to-end full-stack development including user authentication, database design, API development, and payment integration. The platform features both customer-facing interfaces and admin dashboards for product management.

My Role & Contributions

As the sole developer on this project, I was responsible for the complete architecture and implementation. I designed and built the RESTful API backend with Express.js, implementing secure authentication using JWT tokens and bcrypt password hashing. I created all database schemas and models using Mongoose, ensuring efficient data relationships between users, products, and orders. On the frontend, I developed a responsive React application with global state management using Context API, integrated Stripe for payment processing, and implemented protected routes with role-based access control. Challenges & Solutions

Challenge: Authentication Token Management

After implementing JWT authentication, tokens would become invalid after page refresh, logging users out unexpectedly and creating poor user experience.

Solution: Implemented persistent JWT handling by storing tokens in memory and using HTTP-only cookies for sensitive data. Added token refresh logic and middleware to validate tokens on protected routes, ensuring seamless authentication across sessions.
Challenge: CORS Policy Issues

Frontend requests to backend API were being blocked due to CORS errors, preventing communication between React app and Express server.

Solution: Configured Express CORS middleware with proper origin settings and credential handling. Set up Vite proxy configuration for development environment to route API requests through the same domain, eliminating cross-origin issues.
Challenge: Cart State Synchronization

Managing cart state between local storage and database while keeping UI in sync was causing inconsistencies when users logged in or out.

Solution: Implemented a unified cart management system using React Context API that handles both authenticated and guest users. Created merge logic to combine guest carts with user accounts upon login, ensuring no items are lost during authentication.
Challenge: Image Upload Optimization

Large product images were slowing down page load times and consuming excessive storage space on the server.

Solution: Integrated Multer for file handling and implemented image compression before storage. Added file type validation and size limits, then configured CDN-style serving of images to improve load performance.
